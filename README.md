# Adriatic Charters - Luxury Yacht Charter Website

Modern, responsive website za luksuzni najam jahti na Jadranu.

## Kako pokrenuti

1. Otvorite `index.html` u web pregledniku
2. Ili koristite Live Server ekstenziju u VS Code

## Struktura projekta

```
├── index.html      # Glavna HTML stranica
├── styles.css      # CSS stilovi i animacije
├── script.js       # JavaScript funkcionalnost
└── README.md       # Dokumentacija
```

## Značajke

- **Responzivni dizajn** - Prilagođava se svim veličinama ekrana
- **Animacije** - Fade, slide i zoom efekti prilikom scrollanja
- **Carousel** - Automatski rotirajući prikaz ponuda s touch podrškom
- **Lightbox galerija** - Klik na slike za prikaz u punoj veličini
- **Navigacija** - Smooth scroll i mobilni hamburger meni
- **Kontakt forma** - Stilizirana forma za upite

## Sekcije

1. **Hero** - Naslovni prikaz s pozadinom i call-to-action gumbima
2. **About** - Opis usluge i Jadranskog mora
3. **Features** - Kartice s ključnim značajkama
4. **Offers** - Carousel s ponudama (Full-Day, Half-Day, Sunset Cruise...)
5. **Gallery** - Foto galerija s lightbox funkcijom
6. **Contact** - Kontakt informacije i forma za upite
7. **Footer** - Linkovi, socijalne mreže i copyright

## Tehnologije

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display, Montserrat)
- Font Awesome Icons
- Unsplash Images
